大家或许和朋友打过 CF Duel，那种竞技感确实很好玩。但奈何我所在学校打 OI 的人寥寥无几，高达 $0$ 个人能跟我打 Duel。这东西又很好玩，于是我决定自己整一个文化课版本的。

经过简单的检索，我发现市面上的类似平台要么题目过于简单，要么非常不完善，于是我就自己动手做了一个：

**游玩地址**：[lg37.com](http://lg37.com) 
*(备用链接：[math-duel-5191.onrender.com](https://math-duel-5191.onrender.com/))*

### 玩法简介

整体玩法跟 CF Duel 类似，主要针对数学题（因为感觉只有数学/物理才比较适合这种对决形式）。

率先拿到**超过**整张试卷一半分数的队伍即可获胜。

> **吐槽**：最开始试过接入 Gemini 3.1 Pro 实时生成题目，但太贵、太慢，而且 Bug 太多，最后只能降级成了目前的静态题库形式。

题外话：这个英语题面没有致敬 CF 的意思，单纯是找不到太好的中文题库。

### 结语

由于最近比较忙，在这个平台没有出现致命错误的情况下，我可能不会进行频繁的更新和维护了。如果你想改进或参与该项目，可以直接去我的 GitHub 上提交 commit（

当然，如果你觉得这个项目还有点意思，欢迎给我提供 API 赞助或者投喂，谢谢喵~

转眼间，我也快要暂时搁置 OI 了。美好的时间过得好快，可能我校下一届 OIer 不久后也会吐槽没人打 CF 了吧…… 可是如今看来，即使有了数学版本，也鲜有人陪我打 Duel，同学们都忙于课内呢……

最后的最后，祝大家不留遗憾，省选 $\text{RP++}$！
